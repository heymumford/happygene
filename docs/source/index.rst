Features
========

- **Python-first**: Pure Python implementation (no C++ dependencies)
- **Extensible**: Inheritance-based architecture for custom models
- **Realistic**: Hill kinetics, proportional/threshold selection, point mutations
- **Measurable**: Built-in data collection at model, individual, and gene levels
- **Tested**: 100+ tests with 97%+ coverage
- **Fast**: Optimized for populations of thousands across hundreds of generations
